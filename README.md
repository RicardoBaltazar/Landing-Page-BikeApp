### Instalando e iniciando o projeto  
- Primeiro você precisa clonar esse repositório. entre na pasta que deseja instalar o projeto, abra o terminal e execute  o comando: ``` git clone https://github.com/RicardoBaltazar/Landing-Page-BikeApp.git ```    
- Após clonar o repositório, acesse a pasta do projeto e vá para o diretório onde se encontra o arquivo package.json. Abra o terminal nesse diretório e execute o comando: ``` npm install ```  
este comando ira instalar todas as dependências locais que o projeto precisa para funcionar.  
- Após instalar as dependências do projeto, acesse a pasta Pages, abra o terminal e execute o comando:  ``` npm run dev ```  
este comando serve para executar o projeto. Agora você pode acessar o projeto no Browser pelo endereço local: http://localhost:3000/
